# /sb:lesson

```bash
node ~/.claude/skills/sb/commands/_runners/lesson.js $ARGUMENTS
```

Writes `lessons/<date>-<slug>.md`, links back to the source conversation, auto-tags it. Output: the relative path of the new lesson file.

Use this when an insight clicks mid-session and you want to snapshot it without waiting for `/sb:analyze` to batch-process later.
